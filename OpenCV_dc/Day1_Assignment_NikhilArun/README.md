## About Code (pencil_sketch.py)

#Pencil_sketch of any input image
- This code is used to convert any input image into a black and white pencil sketch using blurring and grey scale conversion

#Input parameters
- The code prompts you for the image path and an optional save path for your drawn sketch.

#How to run program
- Click the run button in vs code to run the program. Image path and save path will be prompted as user input.
User can skip the save path prompt by clicking enter.

#Dependencies
- Numpy (numpy)
- OpenCV (cv2)
- Matplotlib (matplotlib.pyplot)

-----------------------------------------------------------------------------------------------------------------------------------------------------------
## Output
- Displays the original and sketch side by side. 
- Saves the sketch if the save location is mentioned.
